Core UI

A canvas where workflow nodes can be selected, dragged, and positioned

Workflow connections

Interaction rules

Clear distinction between creating a workflow step (drop from stencil) and moving an existing step (drag within canvas)

Visual feedback during drag and connect actions, including hover, highlight, and preview connections

Basic editing actions such as select and delete
